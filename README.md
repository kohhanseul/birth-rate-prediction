# Neural Network 기반 출생아 수 예측 (회귀 분석)

## 프로젝트 개요

본 프로젝트는 출생아 수와 관련된 다수의 설명 변수를 활용하여 **출생아 수를 예측하는 회귀 분석 프로젝트**입니다.
단순 선형 관계를 가정하는 모델의 한계를 인식하고, **변수 간 복잡한 비선형 관계를 학습할 수 있는 Neural Network(MLP)** 모델을 적용하여 성능 변화와 학습 특성을 분석하는 데 목적을 두었습니다.

본 프로젝트는 서비스 구현이 아닌, **모델 설계·실험·비교를 통한 분석 역량 강화**에 초점을 둔 포트폴리오용 프로젝트입니다.

---

## 문제 제기

출생아 수는 경제·사회·정책적 요인이 복합적으로 작용하는 지표로, 변수 간 관계가 단순하지 않습니다.

기존의 선형 회귀나 트리 기반 회귀 모델은 해석력은 우수하지만,

* 변수 간 상호작용이 복잡한 경우
* 비선형 패턴이 강한 경우

예측 성능에 한계를 보일 수 있습니다.

이에 본 프로젝트에서는 **각 회귀 모델이 이러한 한계를 얼마나 보완할 수 있는지**를 실험적으로 검증하고, 데이터 규모와 특성에 따른 모델 선택 기준에 대한 인사이트를 도출하고자 했습니다.

---

## 프로젝트 설명

본 프로젝트는 출생아 수와 관련된 다변수 데이터를 입력으로 하여 머신러닝, 딥러닝 기반 회귀 모델을 학습·평가하는 구조로 진행되었습니다.

### 모델링 접근

* 일반 머신러닝 모델과, NN(딥러닝) 모델을 적용하여 성능 비교
* 하이퍼 파라미터를 단계적으로 변경,적용하여 성능 비교
* 각 요소들이 종속변수에 주는 영향을 확인

### 실험 및 분석 포인트

* 하이퍼파라미터 변화에 따른 학습 안정성 및 성능 비교
* Early Stopping 적용 여부에 따른 과적합 방지 효과 분석
* Feature Selection 적용 전·후 성능 차이 비교

실험 결과, **데이터 규모 특성상 Feature Selection을 적용하지 않은 경우가 더 안정적이고 높은 성능을 보였으며**, 이를 통해 변수 선택 기법의 효과는 데이터 크기와 문제 특성에 따라 달라질 수 있음을 확인했습니다.

---

### 담당 역할 (**Neural Network 모델 설계 및 실험**)

* 트리 기반 회귀 모델로 설명되지 않는 **변수 간 비선형 관계를 보완하기 위해 Neural Network 기반 회귀 모델 설계 및 실험 담당**
* 데이터 규모를 고려해 과도한 깊이의 네트워크 대신 **다층 퍼셉트론(MLP) 구조 적용**
* 은닉층 구성, 학습률, 배치 사이즈, 활성화 함수, 옵티마이저 변화에 따른 **체계적인 성능 비교 실험 수행**
* 하이퍼파라미터 튜닝 결과,

  * **학습률 0.01**
  * **은닉층 3개 구조**
  * **Adam Optimizer**
  * **Early Stopping 적용 모델**이 가장 안정적인 성능을 보임
* Feature Selection 적용 여부에 따른 성능 차이를 분석하여,

  * **데이터 규모가 작은 경우 변수 선택이 항상 성능 향상으로 이어지지 않을 수 있음을 확인**

---

> 본 프로젝트는 출생아 수 예측 문제를 통해 Neural Network 회귀 모델의 적용 가능성과 한계를 분석한 포트폴리오용 분석 프로젝트입니다.
